## 1. Install
- 다음 링크에서 다운받는다.

## 2. SSH 연결
- 왼쪽 배너에서 돋보기 클릭
- **Remote Development** 설치
- ```F1``` 누른 후 remotessh 검색
- username@<ip:port> 입력 -> password 입력
